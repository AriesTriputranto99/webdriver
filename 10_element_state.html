Determining Visibility

The following algorithm is used to determine if an element has been displayed.

  • The element must be shown in the viewport in the browser
  • The element has a height and width greater than 0px.
  • The element, or an ancestor, must not have a visibility that is hidden or a display property that is none.
  • OPTIONs and OPTGROUP are treated as special cases, they are considered shown if and only if the enclosing select element is visible.
  • MAP elelemnts are shown if and only if the image it uses is visible. Areas within a map are shown if the enclosing Map is visible.
  • Any INPUT elements of type hidden will not be visibile
  • Any NOSCRIPT elements will not be visible
  • The element should be hidden if their parent has a fixed size AND has the CSS style of "overflow:hidden" and the element's location is not within the fixed size of the parent
